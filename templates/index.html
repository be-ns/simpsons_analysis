Try It

The Simpsons is the longest running TV show ever with 28 seasons to date. However, like many shows, all the different seasons and stories get grouped into a single recommended 'rating' when shown online. Using modern Machine Learning techniques, this project builds a model that predicts the IMDB rating on an episode by episode basis using only the spoken words from the scripts. For a more interactive presentation, I also built a recommender that takes in user-inputted preferences and returns a suggested episode, the predicted rating from my model, and the links to find the video.
